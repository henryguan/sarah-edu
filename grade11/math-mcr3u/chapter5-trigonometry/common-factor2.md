Here is a worksheet with 60 exercises designed to drill the skill of identifying and factoring out the Greatest Common Factor (GCF).

These are categorized by difficulty so you can assign them all at once or break them into sections.

***

# Worksheet: Factoring Trigonometric Expressions
**Objective:** Factor out the greatest common factor (GCF) from each expression.

### Part 1: Numerical Common Factors
*Focus: Factor out the integer coefficient.*

1. $3\sin x + 3$
2. $5\cos x - 5$
3. $2\sin^2 x + 2\cos^2 x$
4. $7\tan x - 7$
5. $4 - 4\sin^2 x$
6. $8\cos x + 16$
7. $-2\sin x - 2$
8. $6\sec^2 x - 6$
9. $9\cos \theta + 9\sin \theta$
10. $10\csc^2 x + 20$
11. $5 - 5\cos^2 x$
12. $12\sin x - 18$
13. $3\tan^2 x + 3$
14. $4\cos^2 x - 4$
15. $15\sin x + 10$

### Part 2: Single Trigonometric Common Factors
Focus: Factor out a single trigonometric term (e.g., $\sin x$).

16. $\sin x \cos x + \sin x$
17. $\cos^2 x - \cos x$
18. $\tan x \sin x + \tan x$
19. $\sin^2 x + \sin x$
20. $\sec x \tan x - \sec x$
21. $\cos x \sin x - \cos x$
22. $\csc x + \csc x \cot x$
23. $\tan x - \tan x \cos^2 x$
24. $\sin x \cos^2 x + \sin x$
25. $\cot x \sin x - \cot x$
26. $\cos x + \cos x \tan x$
27. $\sec x \csc x + \sec x$
28. $\sin x - \sin x \csc x$
29. $\cos x \cot x + \cos x$
30. $\tan x \sec^2 x - \tan x$

### Part 3: Higher Powers & Variable Groups
Focus: Factor out terms with exponents (e.g., $\sin^2 x$) or multiple variables.

31. $\sin^3 x + \sin x$
32. $\cos^4 x - \cos^2 x$
33. $\tan^3 x + \tan^2 x$
34. $\sin^2 x \cos^2 x + \sin^4 x$
35. $\cos^3 x - \cos x$
36. $\sec^4 x + \sec^2 x$
37. $\sin^5 x - \sin^2 x$
38. $\cot^2 x \csc x + \cot^2 x$
39. $\cos^2 x + \cos^2 x \sin^2 x$
40. $3\sin^2 x \cos x - \sin^2 x$
41. $\tan^2 x \sin x + \tan^2 x$
42. $\csc^3 x - \csc x$
43. $\sin^3 x \cos^2 x + \sin x \cos^2 x$
44. $\sec^2 x \tan^2 x + \sec^2 x$
45. $4\cos^3 x + 2\cos^2 x$

### Part 4: Negatives & Tricky Coefficients
*Focus: Factor out negatives or manipulate order.*

46. $-\sin x - 1$
47. $-\cos^2 x - \sin^2 x$
48. $1 - \sin^2 x$ (Hint: View $1$ as $1 \cdot 1$)
49. $-5\tan x + 5$
50. $-\sec x + \sec x \tan x$
51. $-2\cos x - 2\cos x \sin x$
52. $\sin x - \sin^3 x$
53. $-\csc^2 x - 1$
54. $3\sin x - 6\sin^2 x$
55. $-\tan^2 x \sin x - \tan^2 x$
56. $\cos^2 x \sin x - \cos^2 x$
57. $-4\cos^2 x + 4$
58. $\sin x \cos x - 3\sin x$
59. $10\cos^2 x - 5\cos x$
60. $-\sin^2 x \cos^2 x - \sin^2 x$
